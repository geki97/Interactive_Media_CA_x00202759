Components & Features Documentation

This page documents all Bootstrap components and custom JavaScript features implemented in this portfolio website.

Bootstrap Components Used

Navbar

Features Implemented:

  • Fixed navigation bar with responsive collapse button
  • Sticky navbar that remains visible while scrolling
  • Active link highlighting based on current scroll position
  • Smooth navigation to section anchors
  • Responsive toggle for mobile devices
  • Custom styling with gradient background
Grid System

Features Implemented:

  • 12-column responsive grid layout
  • Responsive breakpoints (xs, sm, md, lg, xl, xxl)
  • Flex utilities for alignment (align-items, justify-content)
  • Order utilities for responsive reordering
  • Gutters and spacing utilities (g-4, mb-4, etc.)

Cards

Features Implemented:

  • Hover effects with smooth transitions

Modals

Custom JavaScript Features

Typing Effect

Description: Dynamic typing animation in the hero section that cycles through multiple professional taglines.

Technical Details:

  • Character-by-character typing animation
  • Auto-delete and cycle to next phrase
  • Smooth transitions between phrases
  • Located in hero tagline element
Smooth Scrolling

Description: Enhanced navigation with smooth scroll behavior to section anchors.

Technical Details:

  • Smooth scroll to section IDs
  • Click handler on navigation links
  • Scroll margin offset for fixed navbar
  • Active link highlighting on scroll
Progress Bar Animation

Description: Skills progress bars animate when they come into view using Intersection Observer API.

Technical Details:

  • Intersection Observer for scroll detection
  • Data-percent attribute for target width
  • CSS transitions for smooth animation
  • Unobserves after animation completes
Form Validation

Description: Client-side form validation with success feedback.

Technical Details:

  • Bootstrap form validation styling
  • Email validation using HTML5
  • Required field validation
  • Success message display
  • Form reset after submission
Active Link Highlighting

Description: Navigation links highlight based on current scroll position.

Technical Details:

  • Scroll event listener
  • Dynamic section detection
  • Active class toggling
  • Responsive to all scroll positions
Fade-In Animations

Description: Cards and hobby items fade in as they enter the viewport.

Technical Details:

  • Intersection Observer API
  • Opacity and transform animations
  • CSS transitions for smooth effect
  • Applied to all major content blocks
Tooltips

Description: Bootstrap tooltips on social media buttons in footer.

Technical Details:

  • Bootstrap Tooltip component
  • Data-bs-toggle attribute
  • Title attributes for tooltip content
  • JavaScript initialization
Modal Animations

Description: Smooth animations when modals open and close.

Technical Details:

  • Bootstrap modal events (show.bs.modal)
  • CSS animations on modal content
  • Slide-in-up effect
  • Smooth transitions
Lazy Loading

Description: Images load only when visible in viewport.

Technical Details:

  • Intersection Observer for image detection
  • Data-src attribute for image paths
  • Improves page load performance
  • Graceful fallback for older browsers
Keyboard Navigation

Description: Support for keyboard navigation and accessibility shortcuts.

Technical Details:

  • Escape key to close modals
  • Tab navigation support
  • ARIA labels for screen readers
  • Semantic HTML structure

Additional Features & Design Decisions

Design System
  • CSS Custom Properties (Variables) for theming
  • Consistent color palette (primary: #0d6efd)
  • Professional gradient backgrounds
  • Typography hierarchy with Inter font
  • Consistent spacing using Bootstrap's scale
Accessibility
  • ARIA labels on interactive elements
  • Semantic HTML structure
  • Color contrast compliance
  • Keyboard navigation support
  • Form labels for all inputs
  • Alt text on images
Responsive Breakpoints
  • Mobile: < 576px
  • Tablet: 576px - 768px
  • Small Desktop: 768px - 992px
  • Desktop: 992px - 1200px
  • Large Desktop: > 1200px
  • All sections optimized for each breakpoint
Performance
  • CDN-hosted Bootstrap and Icons
  • Lazy loading for images
  • CSS transitions instead of JS animations
  • Minified CSS and JavaScript
  • Optimized font loading (Google Fonts)
