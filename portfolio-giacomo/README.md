# Giacomo's Portfolio

A professional portfolio website showcasing interactive media projects and creative work.

## Features
- Responsive design with Bootstrap 5.3
- Clean and modern UI
- Project showcase section
- Contact information

## Structure
```
portfolio-giacomo/
├── index.html
├── README.md
└── assets/
    ├── css/

    ├── js/

    │   └── main.js
    └── images/
        ├── profile.jpg
        ├── project-mainline-1.jpg
        ├── project-mainline-2.jpg
        └── placeholder.jpg
```

## Getting Started
1. Open `index.html` in your web browser
2. Replace placeholder images with your own
3. Update content with your personal information
